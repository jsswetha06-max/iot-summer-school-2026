# Smart Plant Watering Monitor

## Components

- ESP32
- Potentiometer (simulated soil sensor)
- DHT11
- OLED
- Red LED (Pump)
- Green LED
- Push Button
- Active Buzzer

## Soil Calibration

Dry Value : 3200

Wet Value : 1200

## Soil States

Dry : <30%

Optimal : 30–70%

Wet : >70%

## Features

✔ Soil moisture monitoring

✔ Temperature & humidity display

✔ OLED status

✔ Automatic watering

✔ Hysteresis

✔ Manual override button

✔ Pump simulation
